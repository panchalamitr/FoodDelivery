# Food Delivery App

I have made food delivery app using [MvRx](https://github.com/airbnb/MvRx) architecture.
Also I have used following things

 - List item

 - Timber

 - Coroutine
 - PaperDb
  - Retrofit
  - EventBus
  - RoundedImageView
